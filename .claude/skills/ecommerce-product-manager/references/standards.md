# E-commerce Standards Reference

## Platform-Specific Standards

### Amazon Standards

#### Listing Quality Standards
| Element | Requirement | Best Practice |
|---------|-------------|---------------|
| Title | 200 characters max | Front-load keywords; include brand + model + key feature |
| Images | 7+ images required | White background main image, lifestyle, infographics, video |
| Bullets | 5 bullet points | Each 500 characters max; lead with benefit, follow with feature |
| Description | 2,000 characters | A+ Content for brand registered; EBC for enhanced visuals |
| Backend Keywords | 249 bytes | No repetition, no competitor names, no ASINs |

#### Performance Standards
| Metric | Minimum | Target | Excellent |
|--------|---------|--------|-----------|
| Order Defect Rate (ODR) | < 1% | < 0.5% | < 0.3% |
| Pre-fulfillment Cancel Rate | < 2.5% | < 1% | < 0.5% |
| Late Shipment Rate | < 4% | < 2% | < 1% |
| Valid Tracking Rate | > 95% | > 98% | > 99% |
| Customer Response Time | < 24h | < 12h | < 4h |

### Shopify Standards

#### Store Performance
| Metric | Acceptable | Good | Excellent |
|--------|------------|------|-----------|
| Page Load Time | < 4s | < 2.5s | < 1.5s |
| Mobile Speed Score | > 30 | > 50 | > 80 |
| Checkout Abandonment | < 75% | < 65% | < 55% |
| Shop Pay Adoption | — | 30% | 50%+ |
